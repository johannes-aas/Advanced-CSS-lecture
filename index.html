The power of inline SVGs

Easy to animate

With SVGs, we can do really cool stuff. Inline SVG code directly in the HTML allows us to change and animate indevidual parts of the element with CSS. Like this light/dark-mode toggle button for example. Go ahead and click on it.

Dash the path!

Creating a CSS drawing animation using dashoffset involves manipulating SVG paths and their stroke properties. The dashoffset property specifies the distance between dashes or gaps in a stroke. By animating this property, you can create the effect of a drawing or revealing animation.
